# vite.svg

**Caminho original:** `src/assets/vite.svg`

## Descrição geral
Arquivo vetorial (SVG) contendo o logotipo oficial do Vite (símbolo em forma de raio/relâmpago nas cores roxo `#9135ff`/`#8900ff` e azul `#00c2ff`). Também é um recurso estático padrão gerado automaticamente pelo template inicial do Vite.

## Conteúdo detalhado
- Formato: SVG, dimensões `77 x 47` (viewBox `0 0 77 47`), com título acessível (`<title id="vite-logo-title">Vite</title>`).
- Composto por múltiplos elementos `<path>`, `<mask>`, `<ellipse>` e filtros de desfoque (`feGaussianBlur`) que compõem o efeito visual estilizado do logotipo, incluindo um estilo interno que adapta a cor dos parênteses (`.parenthesis`) para tema claro (`#000`) ou escuro (`#fff`) via `@media (prefers-color-scheme: dark)`.

## Dependências / Relacionamentos
- Nenhum componente do projeto atual (`App.jsx`, páginas, componentes) importa este arquivo.

## Observações
Resquício do template padrão do Vite/React, sem uso ativo no código-fonte atual. Pode ser removido com segurança ou reaproveitado em telas de créditos/sobre o projeto.
